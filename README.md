4discord
---

Discord bot which fetches random posts from any [4chan.org](https://4chan.org/) board!

### Commands

Every command for the 4discord bot
	
  * `/4help`: Display genereal help for the bot.
  * `/chans <page>`: Display available 4chan boards.
  * `/chan <board>`: Pull a random post from a 4chan board. Not providing a board will randomly pull from /b/!
  * `/chan replies`: Display 5 replies from the latest post pulled by /chan
 
### Background

This bot started as a small, fun project to approach, inspired by another [4chan bot](https://github.com/Romejanic/4chan-Discord-Bot). In the end I decided to carry on a keep adding features and fixing bugs, due to the massive support I received from friends and other people who enjoy reading 4chan content!
